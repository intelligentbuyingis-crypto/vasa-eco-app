# וזה אקולוגיה - מערכת דיגום שדה

## 🚀 Deploy ל-Vercel (חינם)

### שלב 1 - GitHub
1. היכנס ל-github.com
2. לחץ על "New repository"
3. שם: `vasa-eco-app`
4. לחץ "Create repository"
5. העלה את כל קבצי הפרויקט לריפו

### שלב 2 - Vercel
1. היכנס ל-vercel.com
2. לחץ "Add New → Project"
3. בחר את הריפו מ-GitHub
4. לחץ "Deploy" (הגדרות ברירת מחדל טובות)
5. תוך דקה - האפליקציה פעילה!

## 📱 שימוש על טאבלט אנדרואיד
1. פתח Chrome בטאבלט
2. היכנס לכתובת ה-URL של Vercel
3. לחץ על 3 נקודות → "הוסף למסך הבית"
4. האפליקציה תיפתח כמו אפליקציה רגילה!
